# Next.js Sandbox Project

A minimal Next.js 15 project with TypeScript and Tailwind CSS.

## Quick Start

```bash
npm install
npm run dev
```

## Available Routes

- `/test/[id]` - Displays todo item from JSONPlaceholder
- `/api/test/[id]` - Returns test data with timestamp

## Stack

- Next.js 15
- React 19
- TypeScript
- Tailwind CSS
